常见问题

什么是“企业直通班“？

“企业直通班”是莘+学院的核心产品，通过在线教学提升学生技能，保障就业效果。目前覆盖有UI设计、Android、iOS、Python、产品经理、嵌入式等十余个专业方向。选定专业方向并付费后，即可进入班级。每个班级学生不超过30人，有专业教师和班主任进行教学辅导与监督。全程贯穿项目制作来保障学员实践效果。每周会有直播班会进行答疑解惑、项目讨论、模拟面试等内容。最终由就业老师安排就业面试，顺利入职。

什么是LPS呢？

“LPS（Learning Process System的缩写）是莘+学院自主研发的教学过程系统。LPS系统通过整合录播、直播、测试等环节的数据，结合人工智能和数据挖掘技术，提升了老师辅导和同学在线学习的效率和针对性，让学习及就业效果更有保障。LPS目前仍然在不断地进行升级与迭代，只为实现“有教无类，因材施教”的梦想。

莘+学院的视频是免费的吗？

莘+学院官网的课程目前绝大多数是免费开放的。企业直通班的直播视频和各专业的最新视频是仅对企业直通班学员开放的。此外，企业直通班的同学还能够得到专业老师的一对一辅导和讲解、就业老师的就业推荐、班级同学的线上线下活动……

在职人员为什么更适合在线学习？

在职人士学习最关注学习时间和学习效果。如果脱产学习，时间成本和生活压力都很大。在莘+学院企业直通班，即可以使用移动设备随时随地学习，还可以在学业完成、面试通过后再离职。

自律性差，怎么保证完成学习呢？

我们通过改变学习过程，商业项目实战与知识点学习相结合的方式，让学习更有乐趣更有成就感。让同学一旦开始，就想学下去。另一方面，学习是一种习惯，需要自己和老师共同努力进行培养。同学遇到困难或者不适应时，我们会有专业老师和班主任对你进行解惑和监督。

相比线下培训机构，在线学习有什么好处呢？

最直接的好处：记录你的学习进度，通过网络，老师可以更有针对性地解决你的问题；比线下机构给予学生的辅导时间更加宽裕，更能保证你的学习效果。

更自由：我们可以使用电脑、手机等移动设备，随时随地进行学习。

更优质：你可以直接跟随国内一线企业工程师进行学习，经验更丰富，人脉更扎实。

更实惠：我们的性价比更高。因为我们省去了房租成本，提升了老师辅导的效率，所以同样的学习与就业效果，我们的价格更低。

莘+学院怎么保障就业的？

我们通过3种方式解决就业问题：

1.我们和智联招聘、拉勾网、猎聘网直接对接接口，进行岗位推荐和简历推送，让待就业学员最快匹配地点和岗位需求。

2.我们和近千家企业直接签约，保证长期提供指定岗位的人员推荐和培训。

3.每天都会有全国各地的创业型企业、软件园区、投后管理公司主动联系我们，提供招聘需求。

此外，我们还在不断提升自身的教学水平和学习效果，来满足更多企业更高岗位的技能需求，提升学员的就业质量。
